Codex Quick Sheet

키보드 조작 + 자주 쓰는 실행 패턴만 압축

v3 · A4 one-page

Core

  • Ctrl+C stop current run
  • Enter submit
  • ↑ ↓ history / navigation
  • Tab shell completion

One-shot

  • codex exec "task"
  • codex exec --full-auto "task"
  • codex exec --json "task"

Run Modes

  • --full-auto auto-approve in workspace
  • --yolo no sandbox / no approvals
  • exec run then exit

Session

  • codex interactive mode
  • Ctrl+C interrupt turn
  • rerun with edited prompt if needed

Git / Repo

  • run inside trusted git repo
  • git init for scratch dir
  • commit after validation

Validation

  • pytest
  • npm test
  • pnpm lint
  • tsc --noEmit

Diff / Review

  • git diff inspect changes
  • git status changed files
  • review before commit

Good Inputs

  • scope
  • goal
  • validation
  • expected output

Avoid

  • huge vague tasks
  • no validation
  • blind commit / push

Quick Commands

  • codex exec "summarize diff"
  • codex exec "fix failing tests"
  • codex exec "draft PR summary"

Task Shape

  • small refactor
  • bug fix
  • log-based investigation
  • summary draft

One Line

  • Codex는 shortcut보다 실행 패턴과 검증 루프를 같이 보는 쪽이 실제 사용에 더 유리함
