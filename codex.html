Codex 요약 시트

키보드 조작과 자주 쓰는 실행 패턴만 압축

v4 · A4 one-page

핵심

  • Ctrl+C 현재 실행 중단
  • Enter 전송
  • ↑ ↓ 히스토리/이동
  • Tab 셸 완성

원샷 실행

  • codex exec "task"
  • codex exec --full-auto "task"
  • codex exec --json "task"

실행 모드

  • --full-auto 워크스페이스 자동 승인
  • --yolo 샌드박스/승인 없음
  • exec 실행 후 종료

세션

  • codex 대화형 모드
  • Ctrl+C 턴 중단
  • 필요하면 프롬프트 수정 후 재실행

Git / 저장소

  • 신뢰 가능한 git 저장소 안에서 실행
  • git init 임시 디렉토리 시작
  • 검증 후 커밋

검증

  • pytest
  • npm test
  • pnpm lint
  • tsc --noEmit

Diff / 리뷰

  • git diff 변경 확인
  • git status 변경 파일 확인
  • 커밋 전 리뷰

좋은 입력

  • 범위
  • 목표
  • 검증
  • 기대 출력

피할 것

  • 너무 크고 모호한 작업
  • 검증 없음
  • 검토 없는 커밋/푸시

빠른 명령

  • codex exec "summarize diff"
  • codex exec "fix failing tests"
  • codex exec "draft PR summary"

작업 형태

  • 작은 리팩터링
  • 버그 수정
  • 로그 기반 조사
  • 요약 초안
