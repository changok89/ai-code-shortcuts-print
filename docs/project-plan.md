# Project Plan

## Project
AI Code Shortcuts Print

## Goal
Build a small website that shows three printable shortcut sheets on A4-sized layouts:
- Claude Code shortcut
- Codex shortcut
- Gemini Code shortcut

Each sheet should be readable on screen and optimized for printing.

## Core requirements
- A4 print layout
- clean typography
- tool-specific sections
- keyboard shortcuts / command patterns / quick tips
- printable in browser without layout breaking
- responsive enough for laptop/mobile preview

## Deliverables
- static website
- printable A4 pages
- landing/index page to choose tools
- individual printable pages or one combined printable layout
- simple docs for local development and deployment

## Suggested stack
- plain HTML/CSS/JS or minimal static framework
- no backend required
- easy GitHub Pages deployment

## Scope for v1
- landing page
- 3 dedicated shortcut pages
- shared print stylesheet
- copy-friendly layout
- basic iconography or badge-style sections

## Out of scope for v1
- user accounts
- dynamic CMS
- PDF generation backend
- analytics/dashboard
