# Design Plan

## Design direction
- clean developer aesthetic
- white paper feel for print
- dark accent colors per tool
- minimal distractions
- easy visual scanning

## Layout
- A4 portrait first
- header with tool name and one-line description
- grouped sections:
  - essential shortcuts
  - common commands
  - workflow tips
  - do / avoid
- compact footer with version/date

## Tool color ideas
- Claude Code: warm orange / neutral dark
- Codex: green / slate
- Gemini Code: blue / indigo

## Print goals
- `@page size: A4`
- avoid broken cards across pages
- maintain strong contrast in grayscale print
- single-page print target per tool

## UX goals
- easy to read in browser
- print button on each page
- quick navigation between tools
- one-click access to printable version
