Core

  • Enter submit
  • Shift+Enter newline
  • Ctrl/Cmd+C interrupt if supported
  • Tab completion when available

Prompt Shape

  • scope
  • format
  • validation

Useful Requests

  • summarize code path
  • draft plan
  • review diff
  • turn notes into checklist

Output Shape

  • 3 risks

  • validation checklist
  • 5-line summary

Strong Areas

  • summary
  • plan
  • review
  • structured draft

Input Order

  • files
  • logs
  • constraints
  • question

Review Flow

  • summarize first
  • plan second
  • checklist third
  • human approval last

Keep Short

  • one target per prompt
  • one output format
  • one validation rule

Avoid

  • long unstructured context
  • no output format
  • no review step

Mobile

  • read in short blocks
  • use compact outputs
  • prefer bullet summaries

Quick Phrases

  • summarize this
  • draft a plan
  • review risks
  • make checklist
