Gemini Code 요약 시트

단축키보다 요청 형식과 사용 패턴 위주

v4 · A4 one-page

핵심

  • Enter 전송
  • Shift+Enter 줄바꿈
  • Ctrl/Cmd+C 지원 시 중단
  • Tab 가능할 때 완성

프롬프트 구조

  • 목표
  • 범위
  • 형식
  • 검증

유용한 요청

  • 코드 경로 요약
  • 계획 초안
  • diff 리뷰
  • 메모를 체크리스트로 변환

출력 형식

  • 위험 3개
  • 단계별 계획
  • 검증 체크리스트
  • 5줄 요약

강한 영역

  • 요약
  • 계획
  • 리뷰
  • 구조화 초안

입력 순서

  • 파일
  • 로그
  • 제약
  • 질문

리뷰 흐름

  • 먼저 요약
  • 그다음 계획
  • 그다음 체크리스트
  • 마지막에 사람 승인

짧게 유지

  • 프롬프트당 하나의 대상
  • 하나의 출력 형식
  • 하나의 검증 규칙

피할 것

  • 길고 비구조적인 맥락
  • 출력 형식 없음
  • 리뷰 단계 없음

모바일

  • 짧은 블록으로 읽기
  • 압축된 출력 사용
  • 불릿 요약 선호
