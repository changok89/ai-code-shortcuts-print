Claude Code 치트시트

작업 위임, 문서 정리, 범위 있는 구현과 리팩터링에 맞춘 압축형 한 장 요약.

v2 · color / mono toggle

핵심 키

  • Ctrl+C 실행 중단
  • Ctrl+L 화면 정리
  • Ctrl+R 히스토리 검색
  • Ctrl+G 프롬프트 편집기

슬래시 명령

  • /help 도움말
  • /clear 문맥 정리
  • /status 상태 보기
  • /model 모델 전환

자주 쓰는 실행

  • claude --print "task"
  • claude --permission-mode bypassPermissions --print "task"
  • 범위/목표/검증 조건 같이 적기

이럴 때 강함

  • 문서 초안 / 설명 정리
  • 범위 있는 구현 작업
  • 리팩터링 방향 잡기
  • 작업 위임 + 결과 요약

입력 템플릿

  • 범위: 어떤 파일/폴더
  • 목표: 무엇을 바꿀지
  • 검증: 무엇으로 확인할지
  • 출력: 어떤 형식으로 받을지

좋은 습관

  • 검증 명령 함께 넘기기
  • 큰 작업은 단계로 쪼개기
  • 최종 승인 단계 남기기
  • 요약 형식 지정하기

피할 것

  • "알아서 다 해줘" 같은 요청
  • 한 번에 너무 넓은 범위
  • 검증 없이 커밋/배포
  • 파일 범위 없이 추상 지시

작업 위임 패턴

  • 읽기 → 수정 → 검증 → 요약 순으로 지시
  • 문서/설명은 먼저 초안 받고 다듬기
  • 반복 작업은 출력 형식 고정

한 줄 요약

  • Claude Code는 넓은 맥락 이해와 정리, 작업 위임에 강함
  • 범위를 선명히 줄수록 품질이 좋아짐
추천 용도: 작업 위임, 문서화, 범위 있는 리팩터링 출력해 두고 빠르게 참조하기 좋게 압축
