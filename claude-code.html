키보드 단축키만 모은 A4 한 장 치트시트

뒤로
Claude Code

키보드 단축키 전용

설명 없이 단축키만 빠르게 훑는 출력용 요약표

v4 · A4 one-page

핵심

  • Ctrl+C 입력/생성 중단
  • Ctrl+D 세션 종료
  • Ctrl+L 화면 정리
  • Ctrl+O 상세/기록 보기 전환

입력

  • Ctrl+R 히스토리 역검색
  • Ctrl+G 에디터에서 프롬프트 열기
  • Ctrl+X Ctrl+E 에디터 열기
  • Ctrl+V 이미지 붙여넣기

작업

  • Ctrl+B 실행 중 작업 백그라운드 이동
  • Ctrl+T 작업 목록 토글
  • Ctrl+X Ctrl+K 백그라운드 에이전트 종료
  • Esc Esc 되감기 또는 요약

모드

  • Shift+Tab 권한 모드 순환
  • Alt+P 모델 전환
  • Alt+T thinking 전환
  • Alt+O 빠른 모드 전환

여러 줄 입력

  • \ + Enter 빠른 줄바꿈
  • Ctrl+J 줄바꿈
  • / 슬래시 명령
  • ! 직접 bash 실행

멘션

  • @ 파일 멘션 + 자동완성
  • ↑ ↓ 목록 이동
  • ← → 펼치기 / 접기
  • P 미리보기

목록 / 트리

  • R 이름 변경
  • / 검색
  • A 전체 프로젝트
  • B 현재 브랜치

기록

  • / 기록 검색
  • N 다음 일치
  • Shift+N 이전 일치
  • Ctrl+E 전체 보기 토글

기록 종료

  • Q 기록 종료
  • Esc 기록 종료

빠른 참조

  • Ctrl+C 중단
  • Ctrl+L 정리
  • Ctrl+R 히스토리
  • Ctrl+G 에디터

세션 제어

  • Ctrl+B 백그라운드
  • Ctrl+T 작업 목록
  • Esc Esc 되감기
  • Ctrl+D 종료

프롬프트 모드

  • Shift+Tab 권한 모드
  • Alt+P 모델
  • Alt+T thinking
  • Alt+O 빠른 모드
