Claude Code

Claude Code 단축표

작업 위임, 범위 지정, 문서 정리, 리팩터링 지시에 맞춘 한 장 요약표.

v1 · A4 출력용

핵심 단축

  • /help 핵심 명령 보기
  • /clear 현재 대화 문맥 정리
  • /status 모델/세션 상태 확인
  • Ctrl+C 실행 중단

자주 쓰는 명령 패턴

  • claude --print "task"
  • claude --permission-mode bypassPermissions --print "task"
  • 구현 전 파일 범위와 완료 조건을 먼저 명시

작업 흐름 팁

  • 작업 목표, 범위, 검증 기준을 3줄로 나눠 주기
  • 문서/설명 초안은 먼저 받고 최종 마감은 사람이 검토
  • 큰 작업은 기능 단위로 쪼개서 순차 실행

이럴 때 좋음

  • 파일 범위가 비교적 명확한 구현
  • 문서 초안, 리팩터링, 구조 정리
  • 작업 위임과 결과 요약이 같이 필요한 경우

이렇게 쓰기

  • 관련 파일 경로를 먼저 지정
  • 테스트/검증 명령을 함께 넘기기
  • 출력 형식을 지정해서 요약 받기

피할 것

  • "알아서 다 해줘" 같은 모호한 요청
  • 한 번에 너무 넓은 설계/구현 범위 넘기기
  • 검증 없이 바로 커밋/배포로 이어가기
추천 용도: 작업 위임, 문서화, 범위 있는 리팩터링 책상 옆에 두고 바로 보기 좋게 구성
